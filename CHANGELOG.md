## 0.1.0 (2026-07-11)

Initial release.

### Features

- Template codec: encode/decode official GW1 template codes, verified
  against 18 golden fixtures from four independent encoders, fuzzing and
  differential testing
- Build validation: every game rule (attribute budgets, elites, profession
  constraints, hero restrictions) with actionable error codes
- Game data: 1485 skills (Reforged-current) imported from
  @buildwars/gw-skilldata with weekly automated refresh; hero roster
  generated from the GWCA HeroID enum plus a curated metadata overlay
- MCP server: 8 read-only tools, 3 resources, deployed on Cloudflare
  Workers at https://gw1-mcp.graphmaxer.workers.dev/mcp
- GWToolbox AccountExport plugin (C++) with a Windows CI build
